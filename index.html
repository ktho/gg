Instructions For Testing:

In order to test the functions in the terminal, you can use the following commands and paste it into the terminal. First, you can list the current workflows and create a new workflow "A2."

workflow list
workflow create -n "A2" -i "workflow for assignment two"

In order to add nodes to the new workflow, you can use the following commands.

node add -wf "A2" -sn "dad" -t "F" -n "the dad node"
node add -wf "A2" -sn "bob" -t "A" -n "the bob node"
node add -wf "A2" -sn "sis" -t "A" -n "the sis node"
node add -wf "A2" -sn "mom" -t "J" -n "the mom node"

You can link between nodes using the following commands.

link start -wf "A2" -to "dad"
link -wf "A2" -from "dad" -to "bob" -g "bad parenting"
link -wf "A2" -from "dad" -to "sis" -g "bad parenting"
link -wf "A2" -from "bob" -to "mom" -g "favorite son"
link -wf "A2" -from "sis" -to "mom" -g "female bonding"
link finish -wf "A2" -from "mom" -g "the glue"

You can find information about the workflow, nodes, and links through the following commands. For example, the "link children" command shows the children nodes of the given node. Then, you can delete the workflow that you just created with the workflow delete command.

link children -wf "A2" -sn "dad"
node loose -wf "A2"
node list -wf "A2"
workflow delete -n "A2"
